# Refs in React

This application was originally built by **Maximilian Schwarzmüller** on the [React - The Complete Guide 2025 (incl. Next.js, Redux)] course on [Udemy.com].

## Install and Run

Install the dependencies and devDependencies:

```sh
cd react-refs
npm install
```

Start the development server:

```sh
npm run dev
```

This project requires [Node.js] to run. My version at the time was **24.14.0**.
